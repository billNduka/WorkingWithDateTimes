This project demonstrates Java date and time operations with a JavaFX graphical interface.

## Features

1. Display the length of every month in a given year.
2. List all Mondays in a specified month of the current year.
3. Check whether a selected date is Friday the 13th.

The interface is built using JavaFX with FXML and SceneBuilder.

---

## Requirements

* Java JDK 17 or later
* JavaFX SDK installed
* IntelliJ IDEA (recommended)

---

## How to Run

1. Open the project in IntelliJ IDEA.
2. Configure VM options:

```
--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
```

3. Run the class:

```
ui.DateAppUI
```
